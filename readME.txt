## Ejecución del Proyecto

Para ejecutar este proyecto de Java, primero asegúrate de tener instalada la versión 11 de JDK o superior.
Clona este repositorio en tu máquina local y navega hasta el directorio del proyecto. Luego, compila el código fuente utilizando el siguiente comando:

```bash
javac -d bin src/com/tuempresa/tuapp/*.java
